about

Hi, my name is Thordis Kara and I'm a 25 year old Icelander based in copenhagen. I'm currently on my 3rd semester as a Multimedia Design student with emphasis on Digital Design at Copenhagen School of Design and Technology.

I'm interested in everything that has to do with visual design, mobile and web projects as well as branding and visual storytelling.

capabilities

UX design

I draw from user and market research to design prototypes, create user flows, structure information architecture and develop an overall style guide for websites.

Coding

I'm able to build websites using HTML, CSS and JavaScript as well as other frontend frameworks like Tailwind CSS and Motion One.

testing

By testing my prototypes I gain great knowledge about user behavior, wants and needs to deliver solutions that are effective.

strategy

I'm good at creating a digital strategy by aligning a company's brand identity with the desired user experience focusing on every customer touchpoint.
