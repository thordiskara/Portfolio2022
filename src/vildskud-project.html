Vildskud Festival


A promotional website for vildskud festival, an annual independent theater festival in Copenhagen


Vildskud's website

client

Vildskud Festival

role

  • Sustainable redesing
  • Prototyping and testing
  • Visual Identity
  • Web development

ask

To create a mobile first, sustainable website where each page had a maximum size of 0.125 mb. They wanted to explain their ticket buying process in a visual way, have information about the location of their venues and improve the program overview. The design should also resonate with their key values; Creative, Modern and Credible.

solution

As a group we chose a dark color theme as it takes less power to light up darker colors on mobile phones. We mostly used default sans serif font to decrease the pages' size. We created a filtering option for the program to improve it's overview and drew a map so the user's could see the distance form the different venues. We also replaced most images with illustrations, in the form of svgs, so they would take up less space on the web.


My focus area on this project was to develop Vildskud's new visual identity and collaborate on desiging the UI, design events posters and assist with the web development.
