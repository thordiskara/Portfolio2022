about

Hi, my name is Thordis Kara and I'm a Multimedia Design student at Copenhagen School of Design and Technology. I'm currently in my 3rd semester where my main focus has been on digital design. In the program I have learned how to go from ideation, building prototypes and creating design systems to developing websites using HTML, CSS and JavaScript as well as other frontend frameworks.

I'm interested in creating digital solutions, whether it's developing mobile or web solutions, building a digital strategy for clients or doing research and testing.

capabilities

UX design

I draw from user and market research to design prototypes, create user flows, structure information architecture and develop an overall design guide for websites.

Coding

I'm able to build websites using HTML, CSS and JavaScript as well as other frontend frameworks like Tailwind CSS and Motion One.

testing

By testing my prototypes I gain great knowledge about user behavior, wants and needs to deliver solutions that are effective and functional.

strategy

I'm interested in learning how to create efficient and effective digital strategies by aligning a company's brand identity, values and goals with the desired user experience focusing on every customer touchpoint.
